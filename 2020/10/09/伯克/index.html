项目小记 | 关于实时邮件报警项目

背景

产品有泄露测试工位，为了减少报废，需要做一个预警系统，一旦数据组的偏差值大于某个预设阈值，则将邮件发出，报警邮件中配有图片，实时数据组，以及数据离散图示。

软件设计

1. 因为不是一条线，所以需要考虑到多线程

2. 数据采集使用的labview，采集一定量数据，并且在设定量的数据达到目标偏差时，将当前实时数据写入文件。

3. Python进行文件时间戳监控，一旦更新，则将数据读取

4. 考虑到labview统计方法与正常的统计方法有差异，需要使用python进行二次确认

5. 需要2个子程序，一个负责发邮件，一个负责生成图片

继承Threading.thread，使用thread类下run来实现子线程循环运行,判断文件是否更新

class my_thread(threading.Thread):

def __init__(self,threadID,name,path,receiver):
    threading.Thread.__init__(self)
    self.threadID = threadID
    self.name = name
    self.path = path
    self.receiver = receiver

def run(self):
    self.modifydateold = time.strftime('%Y-%m-%d %H:%M:%S',time.localtime(os.stat(self.path).st_mtime))
    while True:
      #  if not os.path.exists(self.path):
      #      break

        print(self.name,' running')
        try:
            self.modifydate = time.strftime('%Y-%m-%d %H:%M:%S',time.localtime(os.stat(self.path).st_mtime))
            if self.modifydateold != self.modifydate:
                print(self.name , self.modifydateold )
                self.send_mail_html(self.name,self.receiver)
                self.modifydateold = self.modifydate
        except Exception as e:
     #       print('watch'+ e)
            logging.info("Thread failed")
            logging.error(e)
            threadlist.remove(self.name)
            break
